フォーム

フォーム部品の値の取得・変更

value属性の値の取得
$('セレクター').val( );
text( )やattr( )のように( )内を空欄にして取得する
お名前を記入してください
お名前 :
$('#btn').click(function( ){
 $("input").val( );
 $('#name').text($("input").val( ));
 });
確認ボタンを押すとinputで取得したものをお名前の欄に書き加える操作をしている
value属性の値の変更
お名前を記入してください
お名前 :
$('btn').click(function( ){
 $('#name').text(' ');
 });
名前を入力した後、クリアボタンを押すことで空欄(' ')に書き換えている
val( )はinput要素だけでなくselect/option要素、textarea要素にも使える

フォームに関するイベント

focus( )
input要素などフォーム部品が選択されると設定された処理を行う
$('セレクター').focus(function( ){
 フォーカスされた時に実行する処理
 });
blur( )
フォーカスが外れた時に設定された処理を行う
$('セレクター').blur(function( ){
 外された時に実行する処理
 });
フォームにカーソルが移ると背景色が変わり、カーソルが外れると背景色が戻る
$('input').focus(function(){
 $(this).css('backgroundColor','pink');
 });
$('input').blur(function(){
 $(this).css('backgroundColor','white');
 });
change( )
フォームの内容が変更されると処理を行う
$('セレクター').change(function( ){
 要素の内容が変更された時に実行する処理
 });
セレクトボックスを選択するとアラートが出る
$('#select').change(function(){
 alert('選択されました!');
 });
submit( )
$('セレクター').submit(function( ){
 要素の内容が変更された時に実行する処理
 });
フォームの送信時に設定された処理を行う。またbutton要素は通常フォーム送信できないがsubmit( )を使う場合は可能
$('form').submit(function(){
 confirm('送信しますか？');
 });

フォーム部品を選択するセレクター

:checked
:checkedはラジオボタンやチェックボタンにチェックが入っている要素だけを選択できる
item1 item2 item3 item4 item5
$('input[type="checkbox"]').change(function(){
 $(':checked');alert('チェックされました')
 });
:selected
:selecedはセレクトボックスで選択された要素だけを選択できる
$('#selected').change(function( ){
 if($(':selected')){
   $('.selectbox').css('color','pink')
  }
 if($('#select').val( ) == ' ' ){
   $('#selectedForm').text( ' ' );
  }else{
   $('#selectedForm').text(`血液型は${$('#select').val( )}です`);
  }
 });
